1.0 Release Candidate 3 - September 20, 2007

This Castle Project version includes the following projects:

  • ActiveRecord - 1.0.3 (RC3)
  • MonoRail - 1.0.3 (RC3)
  • MicroKernel/Windsor - 1.0.3 (RC3)
  • And Common Components, DynamicProxy 1 and 2, Facilities and Services

Distributions

If you are running Windows and would like to use Visual Studio integration, then download the MSI installer, otherwise you download the binaries and/or the source code.

  • Installer version: CastleProject-1.0-RC3.msi

  • Binaries:
    • .net Framework 2.0 debug: Castle-net-2.0-debug-2007-9-20.zip
    • .net Framework 2.0 release: Castle-net-2.0-release-2007-9-20.zip

  • Source code: Castle-SourceCode.zip

  • API Documentation (CHM): Castle-ChmHelp.rar
