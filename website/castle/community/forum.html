» Home » Community » Forum

You can access our forum to ask questions related to the our projects.

Before you post a question, make sure you follow these basic rules:

  • Select the appropriate forum for your question
  • Use a descriptive and meaningful subject. (i.e. do not use "please help me")
  • Ask specific, narrowed, questions. We will not spend an hour to reply to a question
  • Posts like "do my work for me" won't be replied
  • If your problem is an exception, please post the exception type, the message and the inner exceptions
  • If possible compile the version you are using with debug enabled to have more information like line numbers
  • It is always valuable to post code that demonstrates your usage or the scenario
  • When posting code or xml, enclose the content with [code] and [/code] tags

And please remember that being nice is the currency of the open source world.
