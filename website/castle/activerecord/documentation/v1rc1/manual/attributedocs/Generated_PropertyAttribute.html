PropertyAttribute API Document

» Home » ActiveRecord » Documentation » ActiveRecord RC1 Documentation » Reference Manual » Attributes » PropertyAttribute API Document

Maps a standard column of the table.

Example

In the following example, the column is also called 'name', so you don't have to specify. 


public class Blog : ActiveRecordBase
{
    ...
    
    [Property]
    public int Name
    {
        get { return _name; }
        set { _name = value; }
    }

To map a column name, use 


    [Property("blog_name")]
    public int Name
    {
        get { return _name; }
        set { _name = value; }
    }
