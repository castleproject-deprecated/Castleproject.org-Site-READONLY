ActiveRecordValidationBase API Document

» Home » ActiveRecord » Documentation » ActiveRecord RC1 Documentation » Reference Manual » ActiveRecord operations » ActiveRecordValidationBase API Document

Extends Castle.ActiveRecord.ActiveRecordBase adding automatic validation support.

Example


public class Customer : ActiveRecordBase
{
    ...
    
    [Property, ValidateNotEmpty]
    public int Name
    {
        get { return _name; }
        set { _name = value; }
    }
    
    [Property, ValidateNotEmpty, ValidateEmail]
    public int Email
    {
        get { return _email; }
        set { _email = value; }
    }
