PrimaryKeyAttribute

» Main Page » ActiveRecord » Documentation » ActiveRecord RC1 Documentation » Reference Manual » Attributes » PrimaryKeyAttribute

Indicates the property which is the primary key.

Example


public class Blog : ActiveRecordBase
{
    ...
    
    [PrimaryKey(PrimaryKeyType.Native)]
    public int Id
    {
        get { return _id; }
        set { _id = value; }
    }
