PrimaryKeyAttribute API Document

» Home » ActiveRecord » Documentation » ActiveRecord trunk Documentation » Reference Manual » Attributes » PrimaryKeyAttribute API Document

Indicates the property which is the primary key.

Example


public class Blog : ActiveRecordBase
{
    ...
    
    [PrimaryKey(PrimaryKeyType.Native)]
    public int Id
    {
        get { return _id; }
        set { _id = value; }
    }
