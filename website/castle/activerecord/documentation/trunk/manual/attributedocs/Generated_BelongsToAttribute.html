BelongsToAttribute

» Main Page » ActiveRecord » Documentation » ActiveRecord RC1 Documentation » Reference Manual » Attributes » BelongsToAttribute

Maps a one to one association.

Quick Note

Please note that the 'blogid' foreign key lies on the 'Post' table.

Example


public class Post : ActiveRecordBase
{
        ...

        [BelongsTo("blogid")]
        public Blog Blog
        {
        get { return _blog; }
        set { _blog = value; }
        }
