HasManyAttribute

» Main Page » ActiveRecord » Documentation » ActiveRecord RC1 Documentation » Reference Manual » Attributes » HasManyAttribute

Maps a one to many association.

Example


public class Blog : ActiveRecordBase
{
    ...
    
    [HasMany(typeof(Post), RelationType.Bag, Key="Posts", Table="Posts", Column="blogid")]
    public IList Posts
    {
        get { return _posts; }
        set { _posts = value; }
    }
