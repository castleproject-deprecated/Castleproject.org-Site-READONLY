HiloAttribute

» Main Page » ActiveRecord » Documentation » ActiveRecord RC1 Documentation » Reference Manual » Attributes » HiloAttribute

Used when a constraint requires a hilo algorithm

Example


public class Blog : ActiveRecordBase
{
    ...
    
    [HasManyAndBelongs/HasMany,
    CollectionID(CollectionIDAttribute.HiLo),
    Hilo]
    public int Id
    {
        get { return _id; }
        set { _id = value; }
    }
