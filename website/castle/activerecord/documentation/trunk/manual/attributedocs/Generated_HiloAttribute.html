HiloAttribute API Document

» Home » ActiveRecord » Documentation » ActiveRecord trunk Documentation » Reference Manual » Attributes » HiloAttribute API Document

Used when a constraint requires a hilo algorithm

Example


public class Blog : ActiveRecordBase
{
    ...
    
    [HasManyAndBelongs/HasMany,
    CollectionID(CollectionIDAttribute.HiLo),
    Hilo]
    public int Id
    {
        get { return _id; }
        set { _id = value; }
    }
