Sending e-mails

» Main Page » MonoRail » Documentation » MonoRail RC2 Documentation » User's Guide » Sending e-mails
It's easy to send emails using MonoRail combining templates as you do with the views.

Configuring

First of all configure the smtp as described on [[MonoRail Configuration Reference#Formal_reference|Formal reference topic]]

Templates

Create a directory for e-mail templates on your views folder, the directory must be called mail

Processing templates and sending e-mails

The Controller class provides the following methods to handle email:

  1. Message RenderMailMessage(String templateName)

    Creates an instance of Message using the specified template for the body
  2. void DeliverEmail(Message message)

    Attempts to deliver the Message using the smtp configured on web.config
  3. void RenderEmailAndSend(String templateName)

    Renders and delivers the e-mail message.
